# 项目开发输入

**文档版本**：1.0  
**日期**：2026-04-05  
**关联仓库**：TPMS 周期结构生成器 → GRIN（梯度折射率）/ 龙伯透镜扩展

---

## 1. 项目目的

在现有 **TPMS 隐式场 + Marching Cubes + STL 导出** 能力基础上，升级为服务于 **梯度折射率（GRIN）结构设计** 的软件体系，**首个用例为龙伯（Luneburg）透镜**（微波频段、球对称有效介电常数剖面）。

核心设计链为：

**正交周期微结构 + 目标 \(V_f(\mathbf{x})\) 或 \(V_f(r)\)** → **等效介质模型（EMT）** → **目标 \(\varepsilon(r)\)**（球对称 GRIN），并保证 **水密几何模型的体积分数** 与 EMT 所用 **\(V_f\)** 一致（校正闭环）。

---

## 2. 利益相关方与角色

| 角色 | 职责 |
|------|------|
| 结构设计 / 算法 | 隐式晶格、\(V_f(r)\) 驱动、体积校正、指标统计 |
| 电磁 / 射频 | 龙伯剖面、EMT 选型、必要时全波验证（后续阶段） |
| 增材制造（首选 **FDM**） | 最小特征、建造方向、悬垂、各向异性假设 |

---

## 3. 功能范围（In Scope）

1. **GRIN 目标剖面**：由经典龙伯等形式给出 **\(\varepsilon_{\mathrm{target}}(r)\)** 或 **\(n(r)\)**，在 **3～10 层** 径向离散或连续样条近似。  
2. **晶格类型**（分阶段）：以 **shell（TPMS）** 为 P1；在 **隐式场 + \(V_f(r)\)** 框架内 **尽可能丰富** 可选拓扑（TPMS 扩展族、SC/BCC/FCC 等桁架/板系、混合），清单与注册表见 **`docs/03_LATTICE_CATALOG.md`**、**`config/lattice_registry.yaml`**。  
3. **EMT**：基体介电常数可配置（默认见配置），**\(V_f \leftrightarrow \varepsilon_{\mathrm{eff}}\)** 正/反问题（混合律可插拔）。  
4. **几何真实 \(V_f\)**：对 **水密 OBJ/STL** 计算实体体积及 **分壳 \(V_f\)**，与目标迭代直至满足容差。  
5. **质量指标**：以 **最小胞元尺度** 为 **滑动体元**，统计 **径向梯度** 与 **角向波动**（用于评估正交格 + 球向分层导致的穿胞效应）。  
6. **制造约束（嵌入）**：首选 **FDM** 的工艺下限与后验检查（悬垂、最小壁厚等，分阶段落地）。  
7. **运行环境**：**Windows**，单机，可利用 **NVIDIA GPU（如 RTX 4090）** 加速体素/卷积类运算（可选，非首版阻塞项）。

| 参数 | 取值 |
|------|------|
| 频率（首版点频） | **10 GHz** |
| 透镜半径 | **50～150 mm**（配置中给默认与范围） |

---

## 6. 约束与假设

1. **胞元尺寸**：尽量小，**下限由 FDM 工艺**（如喷嘴直径、可靠壁厚/杆径）决定，见工艺配置。  
2. **胞元与波长**：若 **胞元尺度与波长可比**，EMT 仅为 **工程初值**；全波验证列为后续阶段，须在文档与报告中声明。  
3. **正交格 + 球向分层**：层界面可能 **穿过单胞**，名义 \(V_f\) 与离散实现存在偏差；通过 **体积校正** 与 **角向波动指标** 管控。  
4. **FDM 各向异性**：面内与层向介电/力学可能不同；球对称 **\(\varepsilon(r)\)** 为 **目标**，实现上允许 **角向波动** 阈值约束。

---

## 7. 验收标准（分阶段）

### P1（最小可交付）

- [ ] 给定龙伯目标剖面（或离散层目标），输出 **水密 STL/OBJ**。  
- [ ] **整体及分壳** 几何 \(V_f\) 与 EMT 反解目标在容差内（容差数值见规范）。  
- [ ] 配置化：频率、半径、层数、\(\varepsilon_m\)、FDM 关键参数。

### P2

- [ ] 滑动体元报告：**径向梯度**、**角向 \(\sigma\)**。  
- [ ] GPU 加速路径（可选）用于体素统计。

### P3+

- [ ] Truss/plate/hybrid 隐式场；FDM 后验检查增强。

---

## 8. 参考与术语

- **\(V_f\)**：体积分数（实体/设计域）。  
- **EMT**：等效介质理论（Maxwell Garnett / Bruggeman 等）。  
- **GRIN**：梯度折射率。  
- **龙伯透镜**：经典 \(n(r)\) 剖面（具体公式以规范文档选定版本为准）。
