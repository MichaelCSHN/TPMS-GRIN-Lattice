# 晶格类型选项（纳入隐式场 + \(V_f(r)\) 框架）

**目的**：在统一管线（体素标量场 → 等值面/实体指示 → 径向调制 \(V_f(r)\)）下，**尽可能覆盖**文献与工程中常见格型；未实现的条目作为 **registry** 逐项落地。

**统一约束**：每一类须能写成 **\(\mathbb{R}^3\) 上可采样的标量场**（隐式函数、SDF、R-function 组合），以便与现有 **球壳分位数 / iso 调制** 或 **厚度场 \(t(r)\)** 结合。

---

## 1. Shell / 曲面类（TPMS 及推广）

| 代码（建议） | 描述 | 隐式形式要点 | 优先级 |
|--------------|------|----------------|--------|
| P, G, D, I, N | 已实现（见 `TPMS_Mixer_v1.1.0.py`） | 三角函数周期场 \(\Phi\)，\(\|\Phi\|\) 或双等值面成壳 | ✅ |
| FRD, OCTO, I2-Y, N14 等 | 扩展 TPMS / 对称群 | 文献闭式或 Fourier 截断近似 | 高 |
| Schwarz P / D 加厚壳 | 与 P/D 区分：显式双等值面带宽 | \(\Phi\)、\(\Phi\pm t\) 或 \(\|\Phi\|-t(r)\) | 高 |
| 周期极小曲面数值解 | Surface Evolver 等导出系数 | 预存谐波系数 + 栅格求值 | 中 |

---

## 2. Truss / 杆系（隐式近似）

思路：**周期结点图** + **圆柱/胶囊体 SDF** 沿棱；全局 **`min`/`smooth-min`** 并集；\(V_f(r)\) 通过 **杆径 \(d(r)\)** 或 **全局偏移** 调制。

| 代码（建议） | 描述 | 隐式要点 | 优先级 |
|--------------|------|----------|--------|
| SC, BCC, FCC | 立方点阵杆格 | 边集 \(\mathcal{E}\)，\(d(x)=\min_{e\in\mathcal{E}} \mathrm{sdf}_{\mathrm{cyl}}(x,e)\) | 高 |
| Octet, Kelvin 等 | 桁架拓扑名与边集对应 | 同上，换邻接表 | 高 |
| 曲梁 / 变截面 | 工艺友好 | 圆柱半径随 \(r\) 或分段常数 | 中 |
| Body-centered 等混合 | 与 plate 组合 | R-function 融合多 SDF | 中 |

---

## 3. Plate / 板系（隐式近似）

思路：**周期定向平面族** + **带厚板 SDF**（Slabs）；或 **多方向半空间** 交并。

| 代码（建议） | 描述 | 隐式要点 | 优先级 |
|--------------|------|----------|--------|
| SC-plate, BCC-plate, FCC-plate | 文献常见立方板格 | 法向集合 \(\{n_k\}\)，\(\min_k |\langle x,n_k\rangle \bmod \Lambda|\) 成层状 | 高 |
| 菱形十二面体板格等 | 非立方对称 | 换周期与法向集 | 中 |
| 与壳混合 | hybrid | `smooth_min(sdf_{\mathrm{plate}}, sdf_{\mathrm{shell}})` | 中 |

---

## 4. Stochastic / 泡沫（可选）

| 代码 | 描述 | 隐式要点 | 优先级 |
|------|------|----------|--------|
| Gaussian RF | 随机场 + 阈值 | 规则格上相关高斯噪声 → 体素阈值 | 低 |
| Voronoi 泡沫 | 软球堆积 | 到种子距离场 + 阈值 | 低 |

仅当需要 **统计各向同性** 且接受 **非周期** 时使用；与 **GRIN 径向剖面** 叠加需单独定义 **\(V_f(r)\)** 含义（体积约束）。

---

## 5. Hybrid（壳 + 杆 / 板 + 杆）

| 模式 | 公式骨架 |
|------|-----------|
| 加性 / 竞争 | \(\Phi_{\mathrm{hyb}} = \mathrm{smoothmin}(\alpha \Phi_{\mathrm{shell}}, \beta\, \mathrm{sdf}_{\mathrm{truss}})\) 或 R-function |
| 分区 | 按 \(r\) 或按角度切换场（与现有 logistic 混合类似，推广到「格型 A/B」） |

---

## 6. 与 \(V_f(r)\) 的耦合方式（统一接口）

| 参数化 | 适用类型 |
|--------|-----------|
| 单等值面 `iso` 或 \(\|\Phi\|-t\) 中 **\(t(r)\)** | Shell / TPMS |
| **分壳分位数**（当前 `tpms_radial_shells` 思路） | 任意标量场 \(\Phi\) 或 SDF |
| **杆径 \(d(r)\)、板厚 \(h(r)\)** | Truss / Plate |
| **混合权重 \(w(r)\)** | Hybrid |
